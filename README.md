# Pizza-Sales-Report-SQL-Querys

This Project includes a Power Bi Resport and a SQL Querys document, bout try to answer to a pre determinated statments, to identify KPI's and reflet Tendenci charts, from the sales in a Pizza Store.
The Power Bi Resport reflets the insights that will permit identify in a year range (2015), the Total Revenue, Total Orders, the best and worts pizza categories and sizes.


# Used Softwares
- Microsoft Power BI
- Microsoft SQL Server

# Used Stacks
- DAX
- SQL

# KPI's
- Total Revenue
- Average Order Value
- Total Pizza Sold
- Total Orders
- Average Pizzas Per Order

# Chart's
- Daily Trend for Total Orders
- Monthly Trend for Total Orders
- Percentage of Sales by Pizza Category
- Percentage of Sales by Pizza Size
- Total Pizzas Sold by Pizza Category
- Top 5 Best Sellers by Revenue, Total Quantity and Total Orders
- Bootm 5 Sellers by Revenue, Total Quantity and Total Orders
